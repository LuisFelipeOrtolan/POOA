# POOA
Repositório para a matéria de Programação Orientada a Objeto Avançada, ministrada pelo Professor Daniel Lucrédio na Universidade Federal de São Carlos no Ensino Não Presencial  Emergencial (ENPE) 2020/1.

O arquivo noticias.py pode ser considerado o main desse programa. Nele ocorrem as chamadas para as as funções.
O arquvio csv_not.py é o arquivo onde estão as funções ligadas a manipulação de arquivos csv.
O arquivo sites.py é o arquivo onde estão as funções e classes ligadas ao portal de notícias e obtenção de suas notícias, juntamente com os links
O arquvio tratarNoticias.py é o arquivo onde estão as funções relacionadas com o tratamento de notícias vindas de um portal, por exemplo imprimir na tela e colocar as notícias em um arquivo csv.
